# SpringBoot React 게시판(댓글)
### 1.소개
   - 스프링부트(백엔드) 와 리액트(프론트엔드)로 게시판 만들기.
   -게시판 CRUD와 게시글에 댓글 CRD 기능 구현.
   
### 2. 사용기술
  * (서버기술) : SpringBoot, Mybatis, Java , Maven
  * (화면기술) : React, axios, Mui ui
  * (데이터 베이스) : postgresql
